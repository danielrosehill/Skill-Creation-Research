# Compact Research History

Summarise all current individual outputs into a compacted history file for use as context in future iterations.

## Steps

1. Read all files in `outputs/individual/`.
2. Read any existing files in `context/from-history/` to understand what's already been compacted.
3. Create a new compaction file at `context/from-history/YYYY-MM-DD-compaction.md` containing:
   - **Research topic** (from the research brief)
   - **Iterations covered** (which output files were summarised)
   - **Key findings** (consolidated, deduplicated)
   - **Established facts** vs **tentative conclusions** (clearly distinguished)
   - **Contradictions and tensions** (where sources disagree)
   - **Open questions** (what still needs investigation)
   - **Sources consolidated** (unified reference list)
4. The compaction should be comprehensive enough that someone reading only this file would understand the full state of the research.
5. Report the compaction summary and how many outputs were consolidated.
